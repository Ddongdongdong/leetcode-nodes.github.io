
给定一个单链表的头节点 `head`，以及两个整数 `left` 和 `right`，  
要求 **只反转链表中从第 `left` 个位置到第 `right` 个位置的节点**，  
并返回反转后的链表。


dummy + pre + 头插法

本题采用 **局部反转的头插法**，核心思想是：固定反转区间前的节点 `pre`，   让 `cur` 始终指向反转区间的“尾巴”，   每次把 `cur` 后面的节点 **插到 `pre` 后面**。`dummy` ： 虚拟头节点，防止边界问题 ， `pre` : 反转区间前一个节点（锚点） , `cur` :反转区间第一个节点（反转过程中始终是“尾巴”）, `nxt` : 每次要被“头插”的节点 .
代码实现（Python）

```python
class Solution:
    def reverseBetween(self, head: Optional[ListNode], left: int, right: int) -> Optional[ListNode]:
        if not head or left == right:
            return head
        dummy = ListNode(0)
        dummy.next = head

        
        pre = dummy
        for _ in range(left - 1):
            pre = pre.next

    
        cur = pre.next


        for _ in range(right - left):
            nxt = cur.next
            cur.next = nxt.next
            nxt.next = pre.next
            pre.next = nxt

        return dummy.next

时间复杂度：O(n)

空间复杂度：O(1)（原地反转）
